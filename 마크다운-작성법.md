# 마크다운 작성법

## 0. 서문

이 문서는 앞으로 TIL에 적을 문서들의 작성 방식 및 마크다운의 문법을 간단히 소개하기 위해 만들어졌다.  
따라서 자세한 문법은 소개하지 않고, 간결하게 배운 것을 적을 수 있는 수준으로만 설명하도록 하겠다.

## 1. 줄 바꿈

```
맨 뒤에 space 두번을 누르면 줄바꿈이 된다.
이렇게
```

맨 뒤에 space 두번을 누르면 줄바꿈이 된다.

## 2. 제목

```
# 마크다운 작성법
## 제목
```

지금까지 쓰인 두 heading은 위와 같이 #을 사용하면 작성할 수 있다.
